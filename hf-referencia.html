hack-functions 1.4 - Guia de Referência de Funções

por Fernando Mercês <fernando (a) mentebinaria.com.br>

  1. Conversão de base

    • dec2hex - converte um número inteiro em decimal para hexadecimal
    • hex2dec - converte um número em hexadecimal para decimal
    • dec2bin - converte um número decimal em binário
    • bin2dec - converte um número binário em decimal

  2. Conversão de caracteres e strings

    • hex2asc - converte um número em hexadecimal para seu caractere equivalente na tabela ASCII
    • asc2hex - converte um caractere para seu valor equivalente na tabela ASCII, em hexadecimal
    • dec2asc - converte um número em decimal para seu caractere equivalente na tabela ASCII
    • asc2dec - converte um caractere para seu valor equivalente na tabela ASCII, em decimal
    • str2hex - converte uma string para bytes hexa equivalentes a cada caractere (string hexa)
    • hex2str - converte um ou mais bytes hexa em uma string
    • charcalc - converte um ou mais bytes hexa em uma string
    • asciitable - imprime a table ASCII

  3. Criptografia

    • unbase64 - decodifica BASE64 de uma string
    • md5 - calcula o hash MD5 de uma string ou arquivo
    • unmd5 - tenta descobrir a string que gerou o hash MD5 informado
    • rot - encripta/desencripta uma string com a Cifra de César
    • rotall - aplica vários ROTs numa string
    • strxor - faz XOR de uma string com uma chave

  4. Engenharia Reversa

    • asmgrep - busca instruções assembly em binários executáveis

  5. Cáculo numérico

    • xor - calcula o ou exclusivo entre dois números
    • shl - empurra bits para a esquerda de um número
    • shr - empurra bits para a direita de um número
    • pow - eleva um número a uma potência
    • hexcalc - faz cálculo entre números em hexadecimal

  6. Configurações úteis

    • intel - configura o ambiente para disassemblar na sintaxe Intel

str2hex()

Descrição

str2hex <prefixo> <string>

Converte uma string para bytes hexa equivalentes a cada caractere (string hexa).

Parâmetros

prefixo (opcional)
-x   : bytes de saída não espaçados e prefixados com '\x'.
-0x : bytes de saída espaçados e prefixados com '0x'.
-c   : array de bytes no estilo da linguagem C.

string - a string a ser convertida

Saída

Os bytes em hexadecimal correspondentes

Exemplos

$ str2hex "Fernando"
46 65 72 6e 61 6e 64 6f

$ str2hex -x "Fernando"
\x46\x65\x72\x6e\x61\x6e\x64\x6f

$ str2hex -0x "Fernando"
0x46 0x65 0x72 0x6e 0x61 0x6e 0x64 0x6f

$ str2hex -c "Fernando"
{0x46, 0x65, 0x72, 0x6e, 0x61, 0x6e, 0x64, 0x6f}

hex2str()

Descrição

hex2str <bytes_hexa>

Converte um ou mais bytes hexa em uma string. Aceita como entrada todos os formatos de saída da função str2hex.

Parâmetros

bytes_hexa - os bytes a serem convertidos

Saída

A string convertida

Exemplos

$ hex2str '72 6f 63 6b'
rock

$ hex2str '\x72\x6f\x63\x6b'
rock

$ hex2str '0x72 0x6f 0x63 0x6b'
rock

$ hex2str '{0x72, 0x6f, 0x63, 0x6b}'
rock

charcalc()

(hack-functions >= 1.4)

Descrição

charcalc <caractere> <operador> <número>

Efetua cálculos especiais entre caracteres e números.

Parâmetros

caractere - um caractere qualquer ou uma string quando a operação for multiplicação.

operador - +, - ou *

número - um número.

Saída

O(s) caractere(s) correspondente(s)

Exemplos

$ charcalc f + 2
h

$ charcalc B - 1
A

$ charcalc A \* 16
AAAAAAAAAAAAAAAA

$ charcalc isso \* 3
issoissoisso

rot()

Descrição

rot <deslocamento> <string>

Encripta/desencripta uma string com a Cifra de César usando n deslocamentos para a direita.

Parâmetros

deslocamento - o número de deslocamentos a ser aplicado na string

string - a string que sofrerá o ROT

Saída

A string depois do ROT

Exemplos

$ rot 1 'iron maiden'
jspo nbjefo

# simulando rotall()
$ for i in {1..8}; do echo -n "rot$i: "; rot $i 'ewlsd'; done
rot1: fxmte
rot2: gynuf
rot3: hzovg
rot4: iapwh
rot5: jbqxi
rot6: kcryj
rot7: ldszk
rot8: metal

Nota

Essa função possui os aliases rot5(), rot13(), rot18() e rot47(), onde o número de deslocamentos não precisa ser informado, pois já está no nome da função.

strxor()

Descrição

strxor <chave> <string>

Calcula o ou exclusivo de cada caractere de uma string com uma chave.

Parâmetros

chave - o valor, em decimal ou hexadecimal (prefixado com '0x') para efetuar o XOR com os caracteres da string.

string - a string que será 'XOReada'

Saída

A string após o XOR com a chave

Exemplos

$ strxor 4 'ieikjew$ewwewwmjew'
mamonas assassinas

$ strxor 0xa 'pink floyd'
zcda*lfesn

asmgrep()

Descrição

asmgrep <regex> <arquivo>

Busca instruções assembly em binários executáveis e imprime 4 instruções 'em volta'. Você deve atentar-se à sintaxe (Intel ou AT&T) utilizada em seu ambiente.

Parâmetros

regex - a expressão regular (ERE).

arquivo - um binário executável.'

Saída

Caso a expressão regular case, 9 instruções são impressas, sendo 4 anteriores à instrução buscadas e 4 posteriores, além dela mesma.

Exemplos

$ asmgrep 'push *rbp$' /bin/ls
411400:  41 57                          push r15
411402:  41 56                          push r14
411404:  41 55                          push r13
411406:  41 54                          push r12
411408:  55                               push rbp
411409:  53                               push rbx
41140a:  48 83 ec 68                 sub rsp,0x68
41140e:  85 ff                            test edi,edi
411410:  48 8b 9c 24 a0 00 00  mov rbx,QWORD PTR [rsp+0xa0]

hexcalc()

Descrição

hexcalc <número_hexa1> <operador> <número_hexa2>

Faz cálculo entre números em hexadecimal e dá o resultado também em hexadecimal.

Parâmetros

número_hexa1 - número em hexadecimal (com ou sem o prefixo '0x')

operador - +, -, * ou /.

número_hexa2 - número em hexadecimal (com ou sem o prefixo '0x')

Saída

Resultado da operação em hexadecimal (prefixado com '0x')

Exemplos

$ hexcalc 5f7 \* 2
0xbee

$ hexcalc 0xdead / dead
0x1
